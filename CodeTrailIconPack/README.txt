## Inclui
- Ícones `mipmap-*` para launcher
- Adaptive icon (`mipmap-anydpi-v26`)
- Foreground / background / monochrome em `drawable`
- `play-store/icon-512.png`
- Alternativa clara em `alternatives/light/`

## Uso
Substitua as pastas dentro de `android/app/src/main/res/` pelas do pacote.
